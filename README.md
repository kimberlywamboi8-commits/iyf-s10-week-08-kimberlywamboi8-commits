# ✧ Aura Outfit
### The Community Style Journal

**Aura Outfit** is a high-end, minimalist social journal for fashion enthusiasts. Built with React and Vite, it allows users to curate collections, share outfit details, and visualize their aesthetic through a professional, magazine-inspired interface.

---

## ✦ Features
* **Editorial UI:** A "Vogue-inspired" design featuring glassmorphism and premium typography.
* **Outfit Curation:** Users can publish collections with titles, detailed descriptions, and image URLs.
* **Visual Feed:** Dynamic rendering of images with professional "aspect-ratio" handling.
* **Interactions:** Real-time "love" (like) system and post management (delete).
* **Responsive Layout:** A two-column editorial grid that adapts to your style needs.

---

## ✦ Tech Stack
* **Frontend:** React.js (Hooks, Functional Components)
* **Build Tool:** Vite
* **Styling:** Custom CSS3 (Flexbox, Grid, Backdrop-filters)
* **Icons:** Lucide-inspired minimalist glyphs
